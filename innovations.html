#08
Trust Protocol
Mockingbird Handshake™
Read specification → Implement trust layer
The machine-to-machine trust verification protocol for AI content provenance. Mockingbird Handshake™ defines the cryptographic handshake layer that enables AI systems to verify the origin, integrity, and governance status of content before citing it.

Name origin: The mockingbird learns and repeats. The handshake verifies the source before the repeat happens. When an LLM is about to cite content, the Mockingbird Handshake layer provides the verification signal — is this source who it claims to be? Has this content been tampered with? Is there a governance declaration?

First-principles approach: trust must be machine-verifiable, not human-declared.
